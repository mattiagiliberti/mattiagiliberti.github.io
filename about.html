About Me

Programmatore di C++ e Java con accenni di HTML5, CSS3 e JS

Sono Mattia Giovanni Giliberti, studente di 18 anni dell'istituto informatico Blaise Pascal di Reggio Emilia. Prediligo linguaggi di programmazione back-end come C++ e Java. Lato web padroneggio molto bene HTML5, CSS3 e JavaScript: nella creazione di pagine web utilizzo due framework come JQuery e Bootstrap. Tra i linguaggi che conosco aggiungo Qt per la gestione grafica di applicazioni basate su C++ per desktop.

Le mie passioni riguardano principalmente la tecnologia, non sono mai sazio di conoscenze. Ho portato avanti un progetto personale: un sistema operativo. Ho “costruito”, seppur con molte difficoltà, un sistema operativo basato su kernel Linux tramite la documentazione Linux From Scratch (LFS).
